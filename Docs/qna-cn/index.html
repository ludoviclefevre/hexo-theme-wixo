Q Wixo 这名字有何含义？

A Wixo 意思是 A wiki theme for hexo 。

Q 如何指定某些页面生成 ToC(Table of Contents) ？

A 在文章头部添加 toc: true 。

Q 从哪里可以找到你的这几篇文档的 Markdown 源码？

A 在项目的 source 分支。
